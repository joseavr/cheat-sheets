# Git Stash

Stash your commits

```bash
git stash
```

Apply stash

```bash
git stash apply
```

View all stash

```bash
git stash list
```

Drop specific stash at index @{2}

```bash
git stash drop stash@{2}
```

Drop all stashes

```bash
git stash clear
```
